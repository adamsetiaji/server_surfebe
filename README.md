# Surfebe Server

Node.js WebSocket server for Surfebe application with MySQL database.

## Features

- User Management (CRUD)
- Recaptcha Management (CRUD)
- WebSocket Communication
- MySQL Database

## Prerequisites

- Node.js 20+
- MySQL 8.0+
- Docker (optional)
